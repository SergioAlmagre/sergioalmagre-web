Auladex

auladex.com — multi-certification exam platform

Live ↗

Full-stack certification platform with a tokenized economy, AI-driven content pipeline, and real-time community — designed, built, and operated by one developer.

255 API route handlers
1,825 automated tests
126 automation scripts
60 shared contracts
32 lazy-loaded views
308 frontend modules
AI content pipeline — demand-driven course generation via OpenAI in waves of 30 questions. Users vote; when demand crosses threshold, jobs are created and executed automatically through a full lifecycle: draft → validation → active.
Tokenized economy (off-chain) — token ALX earned by completing simulations and streaks. Multi-path acquisition via Stripe EUR, Bitcoin Lightning (OpenNode) and Web3 Polygon. P2P transfers with 1% automatic social scholarship donation and 3% APY staking.
Algorithmic mastery per question — state machine per question (not_seen → in_progress → mastered/weak) computed from accuracy, consistency, and attempt history. Real adaptive logic, not a simple percentage.
Multi-provider, multi-exam — Microsoft (AZ-900 through AZ-400), Docker, HashiCorp, Cambridge English. Provider-agnostic: adding a new certification requires only a catalog entry and a question bank.
RBAC with numeric priority inheritance — admin(400) > operator(350) > business(300) > student(100). Role enforcement at the route layer; every write sealed with SHA-256 payload hashing for cryptographic idempotency.
